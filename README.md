# Library Management System

## Project Overview

The **Library Management System** is a web-based application developed as part of a Software Engineering project. The purpose of this system is to simplify and automate the management of library resources, including books, borrowers, and borrowing transactions.

The system allows librarians to manage books efficiently, while users can browse, borrow, and read books online. It reduces manual record-keeping and improves the efficiency of library operations.

This project was developed using **PHP, MySQL, HTML, CSS, Bootstrap, and JavaScript**.

---

## Project Objectives

The main objectives of this project are:

- To develop a system that manages library books and borrower records.
- To provide a simple interface for librarians to perform book management operations.
- To allow users to browse and read books online.
- To record and track borrowing and returning transactions.
- To demonstrate the application of software engineering principles in a real-world system.

---

## Features

The Library Management System includes the following features:

### User Features
- User registration (Sign Up)
- User login authentication
- Browse available books
- Borrow books
- Return borrowed books
- Read books online

### Admin / Librarian Features
- Login authentication
- Add new books
- Edit book information
- Delete books (soft delete)
- Manage borrower records
- Monitor borrowing transactions

### System Features
- Automated folder creation for book images
- Online book reading functionality
- Organized database storage
- Secure login system

## System Architecture

The system follows a **client-server architecture**.

User requests are processed through a web interface, handled by the PHP application server, and stored in the MySQL database.
